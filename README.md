El sistema a desarrollar permitirá gestionar, principalmente, las ventas y el stock de una empresa fabricante y comercializadora de vehículos que posee una red de concesionarios. 
Para esto contará con un menú principal y una serie de submenús que serán la interfaz para la gestión de los vehículos, las ventas, los clientes, las sucursales y sus vendedores.
